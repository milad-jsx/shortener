# shortener
URL shortener service implemented using NodeJS/express/MongoDB

setup:
    ```npm ci```

test: 
    ```npm run test```

start:
    ```npm run start```

___
__samples__ :
 * Encode:
    * http://localhost:3003/shorter/__www.example.com__
 
 * Decode:
    * http://localhost:3003/revert/__shortenedURL__
